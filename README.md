<h2>Simple Instalation</h2>
```
git clone https://github.com/MF2-akk/Rock-Paper-Scissors.git
cd Rock-Paper-Scissors
```
Then just run the program
```
Python3 main.py
```
<h2>Way with compilation</h2>
```
git clone https://github.com/MF2-akk/Rock-Paper-Scissors.git
cd Rock-Paper-Scissors
pyinstaller --onefile main.py
```
